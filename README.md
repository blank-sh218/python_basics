# Python Basics study

이 폴더는 Python의 문법을 기초부터 정리하고 연습하기 위한 폴더입니다.
각 폴더는 주제별로 구성되어 있으며, 문법 이후에 실습한 코드들로 구성되어 있습니다.

---

## 학습 목표
- Python의 기본 문법과 데이터 구조를 이해하기
- 기초 문법 숙달
- 작은 실습 프로젝트를 병행하면서 코드 실력 키우기

---

## 🗂️ 폴더 구성

| 폴더명 | 주제 | 설명 |
|--------|------|------|
| `01_variables_and_types` | 변수와 자료형 | 숫자, 문자열, 불린형 등 기본 자료형과 연산 실습 |
| `02_control_flow` | 조건문과 반복문 | if, elif, else 조건문과 for, while 반복문 연습 |
| `03_functions` | 함수 | 함수 정의, 매개변수, 반환값, 스코프 이해 |
| `04_classes_and_objects` | 클래스와 객체 | 클래스 정의, 객체 생성, 메서드, 속성 연습 |
| `05_modules_and_packages` | 모듈과 패키지 | import, 표준 라이브러리, 패키지 사용법 |
| `06_file_io` | 파일 입출력 | 파일 읽기, 쓰기, 읽고 쓰기 실습 |
| `07_error_handling` | 예외 처리 | try-except, 예외 종류, 사용자 정의 예외 |
| `08_data_structures` | 자료구조 | 리스트, 튜플, 딕셔너리, 세트 등 |
| `09_standard_library` | 표준 라이브러리 | datetime, os, sys 등 주요 라이브러리 실습 |
| `10_external_libraries` | 외부 라이브러리 | requests, beautifulsoup, numpy 등 설치 및 사용 연습 |
| `11_practice_projects` | 실습 프로젝트 | 미니 계산기, 데이터 처리 등 간단 프로젝트 |
