# Python Basics study

이 폴더는 Python의 문법을 기초부터 정리하고 연습하기 위한 폴더입니다.
각 폴더는 주제별로 구성되어 있으며, 문법 이후에 실습한 코드들로 구성되어 있습니다.

---

## 학습 목표
- Python의 기본 문법과 데이터 구조를 이해하기
- 기초 문법 숙달
- 작은 실습 프로젝트를 병행하면서 코드 실력 키우기

---

## 🗂️ 폴더 구성

| 폴더명 | 주제 | 설명 |
|--------|------|------|
| `01_variables_and_types` | 변수와 자료형 | 숫자, 문자열, 불린형 등 |
| `02_control_flow` | 제어문 | if, for, while 등 |
| `03_functions` | 함수 | 매개변수, 반환값, 스코프 |
| `04_data_structures` | 자료구조 | 리스트, 튜플, 딕셔너리, 세트 |
| `05_file_io` | 파일 입출력 | 파일 읽기/쓰기 실습 |
| `06_modules_and_packages` | 모듈과 패키지 | import, 표준 라이브러리 |
| `07_error_handling` | 예외 처리 | try-except 문 |
| `08_practice_projects` | 미니 프로젝트 | 간단한 문제 풀이 및 응용 |
