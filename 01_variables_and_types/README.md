# 파이썬 프로그래밍의 기초, 자료형

## 학습 목표
- 자료형 이해 및 실습 
- 리스트, 튜플, 딕셔너리를 이해할 수 있다.
- 종합적으로 문제를 풀며 코딩이 가능할 정도이다.

## 실습 내용 
- Numbers_type.py : 숫자형 실습
- String_type.py : 문자열 실습
- String_Formatting.py : 문자열 포메팅 실습

## 핵심 내용
- 자료형 : int, float, str, 
- 연산자 : +, -, *, /, %, **, //
- 복합연산자 : +=, -=, *=, /=, //=, %=, **=
- 문자열 포매팅 : 문자열 안의 특정한 값을 바꿔야 할 경우 쓴다.
- 문자열 포맷 코드 : %s, %c, %d, %f, %o, %x, %%(Literal% 문자%자체)
- 문자열 format 함수 : 문자열 포맷 지정을 더 쉽게 가능
- f문자열 포매팅 : 문자열 앞 f접두사를 붙임
- 문자 개수 세기 : .count()
- 위치 알려 주기1 : .find()
- 위치 알려 주기2 : .index()
- 문자열 삽입 : .join()
- 소문자 -> 대문자 : .upper()
- 대문자 -> 소문자 : .lower()
- 왼쪽 공백 지우기 : .lstrip()
- 오른쪽 공백 지우기 : .rstrip()
- 양쪽 공백 지우기 : .strip()
- 문자열 바꾸기 : .replace()
- 문자열 나누기 : .split()
- 문자열이 알파멧 구성인지 확인 : .isalpha()
- 문자열이 숫자 구성인지 확인 : .isdigit()
- 문자열이 특정 문자(열)로 시작 확인 : .startswith()
- 문자열이 특정 문자(열)로 끝 확인 : .endswith()
